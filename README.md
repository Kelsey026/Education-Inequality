# Education Inequality Trends Across U.S. Metropolitan Areas
**Micro-Internship | Data Analytics & Visualization**

## Overview
This project analyzes changes in **median earnings, unemployment rates, and educational attainment** across U.S. metropolitan areas between 2010 and 2023 using data from the ***U.S. Census Bureau***.  

The goal was to create a **clean, analysis-ready dataset** and develop an **interactive Tableau dashboard** to communicate trends in education-driven inequality.

- **Geography:** U.S. Metropolitan Statistical Areas  
- **Years:** 2010 and 2023  
- **Metrics:** Educational attainment, median earnings, unemployment
